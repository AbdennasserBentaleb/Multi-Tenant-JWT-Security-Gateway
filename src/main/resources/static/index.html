Tenant Gateway

Secure B2B Portal

Log in to experience PostgreSQL Row-Level Security in action.

How it works:

  • Both users authenticate via Keycloak to obtain a JWT.
  • The JWT contains a scoped tenant_id claim.
  • The Spring Boot gateway extracts this claim into a Java 21 ThreadLocal.
  • PostgreSQL RLS physically isolates the data at the database engine level.
